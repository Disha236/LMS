# 📚 Library Management System (C++)

A simple and efficient **Library Management System (LMS)** built using **C++**.
This project demonstrates core concepts of **Data Structures, Algorithms, and STL** by solving a real-world problem of managing library operations.

---

## 🚀 Features

* ➕ Add new books
* 🔍 Search books using **Binary Search**
* 📖 Issue books to students
* 🔄 Return books
* 📋 Display all books
* ❌ Delete books
* 🧠 Efficient data handling using STL

---

## 🛠️ Tech Stack

* **Language:** C++
* **Libraries:** STL (`vector`, `unordered_map`, `algorithm`)
* **Concepts Used:**

  * Structures
  * Sorting (STL Sort + Merge Sort)
  * Searching (Binary Search)
  * Data Management

---

## 🧩 How It Works

```
User Input → Menu → Operation → Data Update → Output
```

* Books are stored using a dynamic **vector**
* Issued books are tracked using **unordered_map**
* Books are sorted by ID for efficient searching

---

## 🧠 Algorithms Used

| Feature      | Algorithm Used     |
| ------------ | ------------------ |
| Add Book     | Sorting (STL sort) |
| Search Book  | Binary Search      |
| Sort Books   | Merge Sort         |
| Issue/Return | Linear Search      |
